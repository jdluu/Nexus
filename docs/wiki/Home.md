# Nexus Wiki

Welcome to the Nexus documentation. Nexus is a cross-platform terminal user interface designed to streamline development workflows by providing a centralized dashboard for tool orchestration and project management.

## Installation

Nexus can be installed directly from the GitHub repository using the `uv` tool manager.

### Standard Installation
```bash
uv tool install git+https://github.com/jdluu/Nexus
```

### Force Update
```bash
uv tool install --force git+https://github.com/jdluu/Nexus
```

## Core Features

*   **Fuzzy Search**: Quickly locate projects using approximate matching.
*   **Persistent State**: Your most recent projects and favorite tools are automatically saved.
*   **Custom Keybindings**: Configure the interface to match your personal workflow.
*   **Theming**: Choose between various modern color schemes (Light, Dark, and Storm).
